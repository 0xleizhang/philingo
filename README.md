## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ or Bun
- Gemini API Key ([Get one here](https://makersuite.google.com/app/apikey))

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/0xleizhang/philingo.git
   cd philingo
   ```

2. **Install dependencies**
   ```bash
   bun install
   # or
   npm install
   ```

3. **Run the development server**
   ```bash
   bun run dev
   # or
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:5173](http://localhost:5173)

5. **Configure API Key**
   Click the settings icon in the header and enter your Gemini API key

## ğŸ“– About

Philingo is named in honor of **Phil**, an inspiring English teacher whose text-based teaching method sparked the creation of this platform. The project evolved from a simple vocabulary lookup tool into a comprehensive language learning platform with AI-powered features.

### The Story

Growing up learning Chinese through reading and recitation, the same approach was brought to English learning by Phil. However, the tedious process of manually looking up words in a dictionary while reading interrupted the learning flow. Philingo was created to solve this problemâ€”providing instant vocabulary help without breaking concentration.

What started as a personal tool has grown to include:
